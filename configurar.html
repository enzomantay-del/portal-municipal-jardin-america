Configuración de Firebase

Asistente para conectar el portal municipal con Firebase (plan Spark gratuito).

← Portal

Paso 1 — Crear proyecto Firebase

  1. Entrá a console.firebase.google.com.
  2. Agregar proyecto → nombre: portal-municipal-jardin-america.
  3. Desactivá Google Analytics si no lo necesitás (opcional).
  4. En el proyecto: Build → Authentication → Get started → Email/Password → activar.
  5. Build → Firestore Database → Create database → modo producción → región southamerica-east1 (São Paulo).
  6. Build → Storage → Get started → misma región.

Paso 2 — Reglas de seguridad

En Firestore → Rules, pegá el contenido de firebase/firestore.rules y publicá.

En Storage → Rules, pegá firebase/storage.rules y publicá. Importante: si ves un selector de bucket arriba, elegí portal-municipal-jardin.firebasestorage.app (debe coincidir con storageBucket en js/firebase-config.js).
